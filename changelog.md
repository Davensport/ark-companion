# Change Log

## v1.0.0 (Stable) - 2023-10-XX
* **Official Release:** formalized versioning system.
* **UI Polish:** Renamed headers and standardized styling.

## v12.2
* **Feature:** Added sorting buttons to the Grid View (Cards).
* **Feature:** List View (Table) is now the default view.
* **Fix:** Breeding tab now force-refreshes data on click to ensure accuracy.

## v12.1
* **Feature:** Added clickable column headers for sorting in List View.
* **UI:** Added dynamic arrow indicators (▲/▼) for active sort columns.

## v11.0
* **Feature:** Added "DELETE LIBRARY" button for mass cleanup.
* **Safety:** Added confirmation dialogs to prevent accidental deletion.

## v2.1 (Python Script)
* **Logic Update:** Script now detects "New" vs "Existing" dinos.
* **Bug Fix:** New dinos default to "Standard" status instead of blank.
* **Safety:** Re-exporting an existing dino no longer overwrites its "Breeder" status.

## v2.0 (Python Script)
* **Core Update:** Switched from Random IDs to Genetic IDs (DinoID1 + DinoID2).
* **Feature:** Enables updating names/stats of existing dinos without creating duplicates.
